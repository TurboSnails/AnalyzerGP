# 投资分析工作流 - 项目计划

## 项目概述

基于你的「投资分析框架」，构建一个自动化分析系统。输入上市公司名称或代码，自动完成从宏观定位到个股深度分析的完整流程，输出结构化的Markdown分析报告。

**支持市场**: A股（沪深）、港股、美股
**数据来源**: akshare（免费）为主，yfinance（美股/港股）补充
**输出格式**: Markdown 分析报告

│   ├── __init__.py

│   ├── stock_screener.py    # 三关过滤（Phase 2）
│   │   ├── drop_checker.py      # 第一关：跌幅量化
│   │   ├── reversal_checker.py  # 第二关：真反转验证

## 分阶段实施计划

### 🔵 阶段一：基础设施（预计 2-3 天）

| 任务 | 说明 | 优先级 |
|------|------|--------|
| 1.1 项目骨架搭建 | 创建目录结构、config.py、requirements.txt | P0 |
| 1.2 市场路由器 | 自动识别股票代码属于A股/港股/美股 | P0 |
| 1.3 A股数据模块 | 用akshare获取行情、财务、估值数据 | P0 |
| 1.4 港股/美股数据模块 | 用yfinance获取基础数据 | P1 |
| 1.5 宏观数据模块 | PMI、M2、国债收益率等 | P1 |

**阶段一交付**: 能跑通 `python main.py 600519`，获取到茅台的基础数据。

### 🟢 阶段二：核心分析引擎（预计 3-4 天）

| 任务 | 说明 | 优先级 |
|------|------|--------|
| 2.1 跌幅量化分析器 | 计算历史高点跌幅、PE/PB分位数、行业对比 | P0 |
| 2.2 真反转验证器 | 毛利率趋势、现金流、库存周期等7维度检查 | P0 |
| 2.3 护城河评分器 | 六维度评分（需结合AI判断） | P1 |
| 2.4 宏观周期定位器 | 美林时钟阶段判断 | P1 |
| 2.5 报告生成器（基础版） | 将分析结果组装为Markdown | P0 |

**阶段二交付**: 能生成包含量化数据的基础分析报告。

### 🟡 阶段三：AI增强分析（预计 3-4 天）

| 任务 | 说明 | 优先级 |
|------|------|--------|
| 3.1 LLM集成基础设施 | 支持调用Claude/GPT/千问等模型 | P0 |
| 3.2 第一性原理分析Agent | 商业模式深度分析 | P0 |
| 3.3 反对清单Agent（空头分析师） | 自动生成反对意见 | P0 |
| 3.4 多头vs空头辩论机制 | 参考TradingAgents的辩论轮次 | P1 |
| 3.5 综合决策Agent | 汇总所有分析，给出评分和建议 | P0 |

**阶段三交付**: 完整的AI辅助分析报告，包含定量+定性分析。

### 🔴 阶段四：优化与完善（持续迭代）

| 任务 | 说明 | 优先级 |
|------|------|--------|
| 4.1 数据缓存机制 | 避免重复请求，提高速度 | P1 |
| 4.2 批量分析模式 | 支持一次性分析多只股票 | P2 |
| 4.3 历史回测 | 用历史数据验证框架有效性 | P2 |
| 4.4 定时监控 | 持仓股票自动跟踪预警 | P2 |
| 4.5 报告美化 | 添加图表、可视化 | P2 |

## 技术选型

| 组件 | 选择 | 说明 |
|------|------|------|
| A股数据 | akshare | 免费、数据全、更新及时 |
| 美股/港股数据 | yfinance | 免费、全球市场覆盖 |
| LLM调用 | litellm | 统一接口，支持多模型切换 |
| 报告生成 | Jinja2 + Markdown | 模板化、可扩展 |
| 命令行 | argparse / typer | 简单易用 |
| 数据处理 | pandas | 标准选择 |

## 使用方式预览

```bash
# 分析单只A股
python main.py --stock 600519
python main.py --stock "贵州茅台"

# 分析港股
python main.py --stock 00700.HK

# 分析美股
python main.py --stock AAPL

# 指定分析深度
python main.py --stock 600519 --depth full    # 完整分析（默认）
python main.py --stock 600519 --depth quick   # 快速筛选

# 批量分析（阶段四）
python main.py --watchlist watchlist.txt
```

**输出示例路径**: `output/600519_贵州茅台_20260416.md`
